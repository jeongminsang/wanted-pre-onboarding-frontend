# 원티드 프리온보딩 프론트엔드 - 선발 과제

## 프로젝트 설치 전
최상위 폴더에 .env 파일 생성 후
```
REACT_APP_API_URL=https://www.pre-onboarding-selection-task.shop
```

또는

./src/utils/todo.ts,

./src/utils/auth.ts

파일 안에 process.env.REACT_APP_API_URL를 "https://www.pre-onboarding-selection-task.shop"로 변경

---
## 프로젝트 설치
```
npm install
```
---
## 프로젝트 실행
```
npm start
```
